# MaiPal - 中医健康陪伴 APP

基于 React + Vite + Tailwind CSS 构建的中医健康管理应用。

## 功能特性

- 👁️ **四诊检测**：望闻问切全面健康评估
- 🤖 **AI 陪伴**：智能对话，个性化建议
- 📊 **数据跟踪**：健康记录和趋势分析
- 🎯 **每日建议**：个性化健康指导

## 技术栈

- React 19
- Vite 6
- Tailwind CSS v4
- React Router
- Motion (动画)
- Supabase (后端)

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
src/
├── app/
│   ├── pages/          # 页面组件
│   ├── components/     # UI组件
│   ├── context/        # React Context
│   └── App.tsx         # 主应用
├── imports/         # Figma 导入资源
└── styles/          # 样式文件
```

## 设计规范

- 字体：思源黑体 (Source Han Sans)
- 栅格系统：8pt
- 主色调：#ECD1B4
- 圆角：12pt
